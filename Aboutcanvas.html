Set Up Your Profile and Notifications

  1. Set up your course profile and add a profile picture.
  2. Configure your notification preferences: Read the Canvas tutorial How do I manage by Canvas Notifications? to set up your notification settings. You will receive notices when something happens in the course (e.g., new announcements, new posts, etc.). So setting your preferences lets you decide how often you want to receive these notices, as well as what personal devices you want to be notified on.
  3. VERY IMPORTANT: Set Announcements to 'ASAP' in case we need to get in touch with you on short notice (you can choose to get Announcements via text or email).
